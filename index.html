It is important to know that despite this being a 'p' element, it is still a block element! It takes up it's own space even if it is inside of other block elements!

Span One Span Two This is to show you that "p" and "span" take the same space and can share lines! But only if you wrap the "span" into the "p"

This is Going to be about Block and Inline elements and the focus of proper structuring of the code!

It is Important to remember these and to use this structure.

  • Use the "header" tag in order to define the header of a section or page.

  • Use the "nav" tag to define the navigation of a page.

  • Use the "main" tag to define the main content of a page.

  • Use the "section" tag to define a section of a page. It is important to note that you can use the "section" tag within other block elements. Examples being; "header", "nav", "main", "footer"

  • Use the "footer" tag to define the footer of a section or page. It is important to note that this is where EVERY major website uses this section for copyright, contact, and other information.

Why is this important?

Here is the answer to that very question:

  • Better accessibility for those who use screen readers!

  • Improves SEO [Search Engine Optimization]

  • Not only is it easier to read and maintain for you, but it also makes it easier to help other developers whom might be sharing the workspace!

Using nothing but the "div" tag can make your code harder to read and maintain, and it can also make it harder for others to understand your code. On top of all of that, it WILL be harder to style in the future.
